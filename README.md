# Plate Detection

Automatic license plate detection in images (and optionally video) using computer vision techniques.

This project locates license plates in images and can be extended to perform OCR to read plate numbers.

---

## About

This repository implements a pipeline for:

- Detecting license plates in images
- Drawing bounding boxes around detected plates
- (Optionally) cropping and saving plate regions
- Serving as a base for plate recognition (OCR)
